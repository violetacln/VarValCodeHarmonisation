
The sql-database table containing all pairings (many-to-one) between variable/values/codes found in px-tables and the harmonised ones has the following columns:

- name_var – variable names found in px-files

- harmon_name_var – harmonised variable name (used in the corrected px-files)

- names_var_values_all – variable values found in px-files

- harmon_name_var_values - harmonised variable values (used in the corrected px-files)

- codes_var_values_all - variable codes found in px-files

- harmon_codes_var_values - harmonised variable codes (used in the corrected px-files)

- title - px-table title

- url – px-table url on Statistics Iceland website

The same notations are used in both R-scripts.
